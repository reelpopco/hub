📘 YouTube Data API v3 키 발급 방법

  1. Google Cloud Console 접속
    👉 https://console.cloud.google.com/ 에 접속합니다.
    구글 계정으로 로그인하세요.
  2. 새 프로젝트 만들기
    상단 바에서 프로젝트 선택 드롭다운 클릭 → 새 프로젝트 클릭
    프로젝트 이름: 아무거나 입력 (예: youtube-finder)
    만들기 클릭 후 잠시 대기
  3. YouTube Data API v3 활성화
    👉 YouTube Data API v3 활성화 페이지 바로가기
    또는: 좌측 메뉴 → API 및 서비스 → 라이브러리 → "YouTube Data API v3" 검색
    사용 버튼 클릭하여 활성화
  4. API 키 생성
    👉 사용자 인증 정보 페이지 바로가기
    또는: 좌측 메뉴 → API 및 서비스 → 사용자 인증 정보
    상단 + 사용자 인증 정보 만들기 → API 키 선택
    API 키가 생성됩니다! 복사해서 위 입력란에 붙여넣기 하세요.
  5. (선택) API 키 제한 설정
    보안을 위해 키 제한을 권장합니다:
    생성된 키 클릭 → API 제한사항 → 키 제한 → YouTube Data API v3만 선택
    이렇게 하면 이 키로는 YouTube API만 사용 가능합니다.

💡 참고: 무료 할당량은 하루 10,000 유닛입니다.
검색 1회 = 100 유닛, 영상 상세 조회 = 1 유닛, 채널 조회 = 1 유닛
일반적으로 하루 50~100회 검색이 가능합니다.
